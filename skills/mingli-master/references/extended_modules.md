# 扩展能力模块

> 这些是紫微本命解读之外的进阶模块。每个模块开头都标明它**属于哪套系统**、
> **能回答什么、不能回答什么**——这是本 skill 的核心原则，绝不含糊边界。
> 见 interpretation_guide 铁律一。

---

## 模块一：择年 · 用田宅宫算"哪年适合买房"

**属于哪套系统**：纯紫微斗数（算时间，是本命的强项）。
**能答**：哪一年置产运最顺 / 最忌。
**不能答**：买哪个区、哪套房、几号楼。

### 脚本（优先用，别手算流年四化）

```
python3 scripts/calculate_horoscope.py --chart /tmp/chart.json --years 2026-2032 --focus property
```
自动逐年算年干四化飞入哪一本命宫，并对【田宅宫】给"★首选 / ○有利 / ✗避开"判断。
合盘买房时对两人各跑一次，找一人得吉化、另一人不犯忌的干净年份。

### 方法（脚本背后的逻辑，供解读参考）
1. 定位本命**田宅宫**在哪一地支。
2. 逐年取流年天干四化（见 four_hua_reference）。
3. 看三吉化 / 化忌是否飞入田宅宫：
   - **化禄入田宅** → 置产最强信号（"喜逢新居"）
   - **化权入田宅** → 主动置产、稳稳拿下
   - **化科入田宅** → 有利，偏改善 / 文书顺
   - **化忌入田宅 → 避开**，房产易破财、纠纷
4. 合盘买房时，**两个人的田宅宫都要算**，找"一人得吉化、另一人至少不犯忌"的干净年份。

### 输出纪律
- 给年份区间 + 每年的吉凶判断表，标清"首选 / 次选 / 避开"。
- 一定要补一句边界："这算的是**哪年**顺，不是**哪里**好；买哪个区看现实因素和房子实物。"

## 模块二：方位 · 八宅命卦算"个人旺方"

> ⚠️ **系统声明（必须对用户明说）**：
> **这是风水（八宅派），不是紫微斗数。** 紫微算时间，不算空间方位。
> 之前若有人拿"命宫地支硬套地理方位"（如命宫在亥→西北），那是错误附会，本 skill 不用。
> 八宅命卦才是风水里算个人方位的正规算法。而且它只是风水的**一派**（还有玄空飞星等，结论可能不同）。

### 脚本（优先用，别手算命卦）

```
python3 scripts/calculate_bazhai.py --year 1995 --gender 女
```
自动求命卦、东四/西四命组、四吉方（生气/天医/延年/伏位）和四凶方。
合盘时对两人各跑一次；若分属不同命组，明说方位冲突，给折中，别假装完美。

### 命卦计算（脚本背后的逻辑，供说明用）
1. 取出生年后两位数字相加，反复相加到个位数。
   （例：1995 → 9+5=14 → 1+4=**5**）
2. **1900–1999 出生**：
   - 男命：`10 − 得数`（结果为 5 时作 2 坤）
   - 女命：`得数 + 5`，再反复相加到个位（结果为 5 时作 8 艮）
   （例：女 1995 → 5+5=10 → 1 = **坎卦**）
3. **2000 年及以后**出生：
   - 男命：`9 − 得数`（为 5 作 2 坤）
   - 女命：`得数 + 6`，反复相加（为 5 作 8 艮）

### 命卦 → 命组 → 吉凶方位
- **东四命**（坎1、离9、震3、巽4）：吉方 = 东、东南、南、北
- **西四命**（乾6、坤2、艮8、兑7）：吉方 = 西、西北、西南、东北

四吉方细分（可用于精细建议）：
| 吉星 | 含义 |
|------|------|
| 生气 | 最旺——事业、财运、活力 |
| 天医 | 健康、贵人、稳定 |
| 延年 | 感情、婚姻、人际和谐 |
| 伏位 | 平稳、积蓄 |

四凶方（避）：绝命、五鬼、六煞、祸害。

### 落地到城市区域（可选）
把吉方对应到目标城市"相对市中心的方向"，给区域参考。
**但必须补上边界**："这是大方向筛选。一套房子旺不旺，最终看它本身的坐向、门向、采光、周边地形——命卦给不了这么细。"

### 合盘方位
- 分别算两人命卦。
- 若**同命组**（都东四或都西四）→ 方位天然一致，好办。
- 若**跨命组** → 明说存在冲突，给折中建议（例：取两人吉方的交集，或以主要居住/工作者为主），不假装完美。

## 模块三：合盘 · 两人契合度

**属于哪套系统**：紫微斗数（用两张本命盘互参）。
**能答**：性格契合、感情模式互动、大方向合不合。
**不能答**：这段关系"能不能成"的定论——关系是活的，命盘只给底色。

### 方法
1. 各自排本命盘。
2. 比对关键宫位的呼应关系：
   - **命宫 ↔ 命宫**：性格底色合不合、气场是否相冲（地支相冲要点出）
   - **各自夫妻宫**：对方是否符合自己夫妻宫描述的伴侣类型
   - **一方命宫 ↔ 另一方夫妻宫**：契合度的关键信号（一方的命落在另一方期待的伴侣位）
   - **五行/大限**：两人当前十年运是否同步、互补还是互耗
3. 给分维度参考（命宫契合 / 夫妻宫 / 五行 / 大限同步），但**说明这是象征性评分，不是科学量表**。

### 输出纪律
- 先讲合的地方，再讲需要磨合的地方，最后强调"命盘给底色，经营靠两个人"。
- 不下"你们注定在一起 / 注定分开"这种判决（呼应铁律三）。

## 模块四：择日 · 领证 / 婚礼 / 开业选日子

> ⚠️ **系统声明 + 能力边界（必须对用户明说）**：
> 择日属**黄历**范畴。本 skill **只给到"月份窗口 + 大致时段"，不声称精确到某年某月某日**。
> 原因：手工推算某天的干支（判断冲煞、择吉），可能有 **1–2 天误差**；
> 真正精确到日的黄历，需要专业万年历/择日历核对。**装作能精确到天，就是不诚实。**

### 能答 / 不能答

- **能答**：哪几个月份对两人相对有利（结合流年四化、双方生肖冲合的大方向）。
- **不能答**：精确到"X 月 X 日几点"的黄道吉日——那需要万年历，不在紫微/本 skill 能力内。

### 方法（只到月份粒度）

1. 用 `calculate_horoscope.py` 看当年两人的流年四化，避开化忌冲夫妻宫/命宫的月份大方向。
2. 结合双方**生肖**避开明显相冲的月支（如属狗者避辰月、属猪者避巳月的粗筛）。
3. 给出 **2–3 个候选月份窗口**，每个窗口说明为什么相对有利。

### 输出纪律（务必照做）

- **给月份，不钉死某天。** 措辞如："建议在 X 月这个窗口内，具体某天请用万年历挑黄道吉日。"
- **主动交代边界**："命理能帮你选到月份这个粒度；到某一天的冲煞吉凶，要请你用万年历或专业择日核对，我不糊弄你精确到天。"
- 绝不把手工推的干支当成确定结论丢给用户（呼应铁律一、铁律二）。

## 通用纪律（所有扩展模块共享）

1. **每个模块开头先亮系统归属**——紫微 or 风水 or 黄历，不混着糊弄。
2. **算不了的直接说算不了**，不为了显得全能而附会。
3. **黄历择日**（具体到某天）若使用，务必提示："手工推算干支可能有 1–2 天误差，请与万年历核对。"
