# 四化飞星参考

> 分析四化时查阅。四化是紫微斗数的"动态引擎"——星是静态禀赋，四化是能量的流向。
> 这是本 skill "算时间"的核心武器：大限四化定十年，流年四化定当年。

## 四化是什么

每个天干引动四颗星产生变化：**化禄、化权、化科、化忌**。

| 四化 | 性质 | 白话 |
|------|------|------|
| **化禄** | 财、缘、顺、增益 | 好事来了、路顺了、有进账 |
| **化权** | 权、力、掌控、扩张 | 拿到主导权、能力被放大、做事有劲 |
| **化科** | 名、贵人、文书、化解 | 名声好、有贵人、文书顺、逢凶稍解 |
| **化忌** | 阻、耗、执、是非 | 卡住了、消耗、放不下、生是非 |

**禄权科是三吉化，忌是唯一的凶化。** 但化忌不全是坏——它代表"你最在意、最放不下、能量最集中"的地方，是课题所在。

## 怎么用四化算运（本 skill 的看家本领）

### 第一步：确定天干
- **大限四化**：用大限宫位的天干 → 定这十年的主旋律
- **流年四化**：用流年地支对应的天干 → 定这一年的重点

### 第二步：找四化星飞入哪个"本命宫位"
关键动作：**流年/大限的化星，落在本命盘的哪一宫**，就影响那件事。

- 化禄飞入 → 那个领域今年有增益、机会、进账
- 化权飞入 → 那个领域你今年掌控力强、主动、扩张
- 化科飞入 → 那个领域今年有名声、贵人、文书顺
- 化忌飞入 → 那个领域是今年的课题、堵点、是非源

### 第三步：解读

**范例（本 skill 实战逻辑）：**
> 某女大限天干己 → 己干四化：武曲禄、贪狼权、天梁科、文曲忌
> - 武曲化禄飞入官禄宫 → 这十年事业收入稳步上升 ✓
> - 贪狼化权飞入命宫 → 个人魅力、主导力极旺 ✓
> - 文曲化忌飞入夫妻宫 → **感情是这十年最大课题**，易有"说不清的裂缝"或被花言巧语迷惑 ⚠️
> 一句话：事业开挂，感情是考场。

**买房/置产专用**：看流年三吉化是否飞入**田宅宫**
- 化禄入田宅 → 置产最强信号（"喜逢新居"）
- 化权入田宅 → 主动置产、稳稳拿下
- 化科入田宅 → 有利，偏改善/文书顺
- **化忌入田宅 → 避开**，房产易破财纠纷

**婚姻/感情专用**：看流年/大限化星飞入**夫妻宫**
- 三吉化入 → 感情有进展、时机好
- 化忌入 → 感情课题浮现，需谨慎

## 化忌的特别提醒（诚实命理师的态度）

化忌不等于"完蛋"。它是能量最集中、最放不下的点。
- 化忌入某宫 = 你今年会在这件事上**投入最多、也最容易卡住**
- 解法不是恐惧，是**觉察**：知道能量堵在哪，就能主动疏导

**绝不能**用化忌吓人（"你今年有大劫"），那是恐惧生意的套路。
正确说法：**"这一年这块是你的课题，主动面对它，别让它积到爆发。"**

## 自化与飞化（进阶，可选）

- **自化**：宫干引动本宫星曜四化，主"自己招来的"变化
- **飞化**：宫与宫之间的四化牵连，看宫位间的因果关系
进阶技法，日常解读用不到时可略过，不必为了显得高深硬塞。
